문서구조

DTD

<!DOCTYPE HTML>

이 사이트는 HTML5 표준 문서로 작성

기본언어

<html lang="ko">

기본언어는 한글이며, 해당 국가별 언어코드로 변경

기본 문자셋

 <meta charset="utf-8">

기본 문자셋은 UTF-8 입니다.

IE 브라우저 호환성

<meta http-equiv="X-UA-Compatible" content="IE=edge">

인터넷 익스플로러에서 해당브라우저의 최신 렌더링 엔진으로 표시될 수 있게 강제 변경(IE10 이상)

모바일 용 뷰포트

<meta name="viewport" content="width=device-width, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no">

모바일 단말기에서 사용자가 임의로 화면을 확대/축소 할 수 없고, 기본 너비는 사용자 단말기 너비 지정

기술기준

Css

css Level 3을 준수하고 기준에 준수하여 작성

문서 표현을 위한 기본 정의, 레이아웃, 반응형을 위한 media query 등을 정의하고 있습니다. 기본적으로, 레이아웃 구성과 동적 인터렉션을 위해서 ID 셀렉터를 사용하고 정적 디자인 표현을 위해서는 class 셀렉터를 사용합니다. 셀렉터 네이밍은 SMACSS, BEM명명법을 혼용

Javascript

jQuery version : v1.10.2 사용

자바스크립트 표준안 ECMAScript 5.1 기준에 준수하여 작성

반응형 웹 브레이크 포인트

720이하(Mobile), 721이상 1023이하(Tablet : Mobile View), 1024이상 1279이하(Tablet : Web View), 1280이상(Web), 1680이상(Wide-Web)으로 분기하여 media query를 사용
